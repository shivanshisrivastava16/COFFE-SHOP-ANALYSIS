# COFFE-SHOP-ANALYSIS
☕ Unlocking Coffee Shop Insights: A Power Pivot Dashboard Project

I'm excited to share a recent project where I transformed raw coffee shop transactional data into an interactive and actionable business intelligence dashboard using only Microsoft Excel and Power Pivot.

Managing a dynamic business like a coffee shop requires real-time insights. By leveraging the data modeling capabilities of Power Pivot, I was able to handle complex relationships between datasets and deliver an intuitive interface for decision-makers.

🔑 KEY FEATURE AND INSIGHTS:

This dashboard provides a comprehensive overview of performance through multiple lenses:

EXECUTIVE KPIs:
A quick snapshot of total performance, showing Total Sales (~$699k), Total Footfall (~149k), and key efficiency metrics like average bill per person.

Order Trend Analysis:
Pinpoints peak operational hours. The data clearly shows a major peak around 10:00 AM with nearly 3,000 orders, and a sharp drop in activity after 6:00 PM.

Sales Performance by Day:
Uncovers sales patterns throughout the week. Interestingly, Tuesdays emerge as the top-performing day for sales, followed by Mondays and Sundays, highlighting an early-week strength.

Product Type Breakdown:
Identifies key revenue drivers. Top product types by sales include Barista Espresso, Brewed Chai Tea, and Hot Chocolate.

Location Comparison:
Compares performance across different store locations. The Astoria and Hell's Kitchen locations are nearly neck-and-neck for top sales.

Full Interactivity: 
Powerful slicers for Day and Month allow for granular analysis of specific time periods.

🛠️ Tools & Techniques
Excel Power Pivot: For data modeling and handling large datasets with complex relationships.

DAX (Data Analysis Expressions): Used to create powerful measures for calculating KPIs and driving visualization logic.

Slicers & Timelines: For creating an intuitive and dynamic user interface.
